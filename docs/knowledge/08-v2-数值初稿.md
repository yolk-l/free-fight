# v2 数值拍板（调参单一事实来源）

> **状态**：**v2.1 重平衡已实现**（2026-05）；目标游戏时长 3-5 分钟。  
> **前提**：持仓负面线性叠加、无 soft cap；弃牌无代价；冷却弃 1 张；手牌上限 **7**；即时制。

---

## 1. 设计意图

| 体验目标 | 数值手段 |
|----------|----------|
| 游戏时长 3-5 分钟 | 英雄足够强应对前期，中后期逐渐不可持续 |
| 留牌有压力，但能短留 1-2 张轻卡做规划 | 史莱姆/蝙蝠税最轻 |
| 哥布林「拿着疼、打出值」 | 哥布林税最重 + 失血 |
| 满手乱囤会崩 | 7 张叠满故意很痛，逼部署/弃牌 |
| 弃牌是减压阀，不是主玩法 | 冷却 10s，无代价 |
| 部署后主要风险仍是场上怪 | 怪物有难度梯度 |
| 装备有成长感 | 5 种装备覆盖不同策略 |

## 9. 3-5 分钟生存模拟

**快速部署策略**（拿到卡就放）：
- 每 3s 来一张卡，英雄平均 2-3s 杀一只
- 同时面对 1-2 只怪，平均承受 ~4-6 DPS
- 150 HP 可撑 ~25-37s per 怪群
- 装备逐步提升后可持续更久
- 预计存活 4-5 分钟

**留牌策略**（留 2-3 张再部署）：
- 持仓负面降低有效属性
- 中期防御下降后怪物 DPS 提高
- 预计存活 3-4 分钟

**不管理策略**（堆满手牌）：
- 失血 + 属性削弱 → 1-2 分钟内死亡
