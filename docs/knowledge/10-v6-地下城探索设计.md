# v6 玩法设计：地下城探索

## 设计背景

v5 的"部署在哪"决策轴较弱——Voronoi 地形不直观，大部分玩家会随意放置怪物。v6 将战场从"自由竞技场 × 4 场"改为"单张大地图格子探索"，让每次部署 = 规划英雄的行走路径，每一步都有意义。

## 核心约束（不变）

- 玩家唯一操作 = 从3张候选选1张拖到格子上
- 无主动技能、无弃牌、无装备
- 怪物只由玩家部署产生（Boss除外）
- 线框 Demo 定位

---

## 1. 核心循环

```
玩家看到3张候选 → 选1张拖到已探索的格子上
→ 怪物出现在该格子（固定不动）
→ 英雄沿最短路径走向该怪物
→ 每经过一个格子，触发该格子效果（加血/陷阱/随机事件...）
→ 到达怪物格，进入战斗（实时近战，与v5相同）
→ 击杀后自动获得属性+共鸣
→ 英雄寻找场上下一个最近的怪物，继续走
→ 场上无怪时英雄原地等待
→ 2s CD后刷新3张新候选
→ 循环直到击杀Boss或英雄死亡
```

**策略核心**：玩家不直接控制英雄移动，而是通过"在哪个格子放怪物"来间接规划英雄的行走路线。路线经过的格子效果决定了英雄的成长和存亡。

## 2. 地图结构

### 2.1 整体布局

```
┌─────────────────────────────────────────────┐
│                                             │
│   [起点]──走廊──[房间A]──走廊──[房间B]      │
│                    │                  │      │
│                  走廊              走廊      │
│                    │                  │      │
│              [房间C]──走廊──[Boss房间]        │
│                                             │
│           迷雾覆盖未探索区域                  │
└─────────────────────────────────────────────┘
```

- **地图尺寸**：30×20 格（大地图，需要相机跟随）
- **结构**：房间 + 走廊，随机生成（类 Roguelike 地牢）
  - 3-5 个房间（5×5 ~ 8×8 大小），由 2-3 格宽的走廊连接
  - 房间内有密集的功能格
  - 走廊以空地为主，偶有陷阱/问号
- **起点**：左下角区域，英雄在此出生，周围 3×3 范围已探索
- **Boss**：右上角区域的 Boss 房间，地图上有方向指示箭头
- **墙壁**：不可通行、不可部署的格子，形成走廊和房间
- **每格大小**：64×64 像素

### 2.2 迷雾系统

- 未探索的格子被黑色迷雾覆盖，看不到内容
- 英雄视野半径：3 格（曼哈顿距离）
- 英雄移动时实时揭开迷雾
- **玩家只能在已揭开的空格子上部署怪物**
- 迷雾一旦揭开永久保持（不会重新覆盖）
- Boss 房间方向在地图边缘有箭头指示，但内容未知

### 2.3 相机

- 相机跟随英雄，平滑插值
- 玩家可拖拽/滚轮缩放查看已探索区域（规划部署位置）
- 缩放范围：0.6x ~ 1.5x
- 小地图在右上角，显示已探索布局 + 英雄位置 + Boss方向

## 3. 格子类型

### 3.1 总览

| 类别 | 格子 | 视觉 | 经过时触发 |
|------|------|------|-----------|
| **空地** | 普通地面 | 浅灰 | 无效果 |
| **正面** | 治愈泉 | 浅蓝 | 英雄回复 10 HP |
| **正面** | 力量祭坛 | 红色 | 英雄获得攻击+2（临时 10s） |
| **正面** | 铁壁祭坛 | 银灰 | 英雄获得防御+2（临时 10s） |
| **正面** | 共鸣水晶 | 紫色 | 下次击杀共鸣进度 ×2 |
| **正面** | 宝箱 | 金色 | 随机永久属性 +1（攻/防/攻速/HP） |
| **负面** | 毒沼 | 暗绿 | 英雄受 8 伤害 |
| **负面** | 陷阱 | 暗红 | 英雄受 12 伤害 |
| **负面** | 诅咒地 | 暗紫 | 英雄攻击-2（临时 10s） |
| **负面** | 减速泥 | 棕色 | 英雄移速减半（临时 8s） |
| **随机** | 问号格 | 黄色? | 从事件池随机抽取一个效果 |
| **特殊** | 视野塔 | 白色 | 揭开周围 6 格范围迷雾 |
| **特殊** | 传送阵 | 蓝紫 | 传送到地图上另一个传送阵 |
| **特殊** | Boss门 | Boss色 | Boss 房间入口（进入后触发 Boss 战） |

### 3.2 格子分布规则

- **起点周围（5格内）**：70% 空地、25% 正面、5% 问号，无负面
- **中间区域**：50% 空地、20% 正面、15% 负面、15% 问号
- **Boss 附近**：40% 空地、15% 正面、25% 负面、15% 问号、5% 宝箱
- **走廊**：80% 空地、10% 问号、10% 负面
- **房间**：格子密度高，正面/负面/问号集中

### 3.3 问号格事件池

| 事件 | 权重 | 效果 |
|------|------|------|
| 祝福 | 3 | 回复 15 HP |
| 力量涌现 | 2 | 攻击永久 +1 |
| 坚韧 | 2 | 防御永久 +1 |
| 灵敏 | 2 | 攻速永久 +0.1 |
| 诅咒 | 3 | 受 10 伤害 |
| 虚弱 | 2 | 攻击-3（临时 15s） |
| 迷雾扩散 | 1 | 周围 4 格重新被迷雾覆盖 |
| 共鸣脉冲 | 2 | 所有共鸣路径 +1 进度 |
| 精英召唤 | 1 | 随机一只怪在附近空格生成（精英版） |

格子效果触发后，该格子变为空地（一次性效果）。
视野塔和传送阵保持存在。

## 4. 英雄移动

### 4.1 寻路

- 使用 A* 寻路（Godot 内置 AStar2D）
- 英雄走最短路径到目标怪物
- 移动方式：逐格移动，每步之间平滑插值
- 移动速度：每格 0.3 秒（约 213 px/s，与当前 180 px/s 接近）
- 每到达一个新格子 → 触发该格子效果 → 继续移动
- 对角移动：不允许（四方向移动，更清晰可读）

### 4.2 目标选择

- 英雄始终走向场上最近的怪物（A* 路径距离，非直线距离）
- 击杀后切换到下一个最近的怪物
- 场上无怪物时：英雄原地等待
- 英雄走向怪物途中，如果玩家部署了更近的怪物：**不切换**，完成当前目标后再重新选择
  （避免英雄频繁改变路线导致路径不可预测）

### 4.3 路径预览

**关键交互**：玩家拖拽候选卡在地图上悬停时，显示：
- 从英雄当前位置到该格子的预计路径（虚线）
- 路径经过的格子高亮，附效果图标
- 路径格数 + 预计行走时间
- 该格子的地形效果提示

这是"部署在哪"策略性的核心 UI 支撑。

## 5. 部署规则

### 5.1 可部署范围

- 只能部署在**已探索的、空的、可通行的**格子上
- 不能部署在：墙壁、已有怪物的格、英雄所在格
- 不能部署在迷雾覆盖的格子上
- 拖拽时不可部署的格子显示红色 ×

### 5.2 部署与CD

- 与 v5 相同：选1张 → 其余2张消失 → 2s CD → 刷新3张
- NextHandPreview 在 CD 期间显示下一批

### 5.3 怪物在格子上

- 部署后怪物固定在该格子上，不移动
- 怪物占据格子，英雄走到相邻格时开始攻击（攻击范围 = 1格 = 64px）
- 怪物死亡后该格子变为空地
- 怪物死亡机制保留（分裂的小史莱姆出现在相邻空格）

## 6. Run 结构变化

### 6.1 从 4 场到 1 张地图

| v5 | v6 |
|----|----|
| 4 场独立战斗 | 1 张大地图持续探索 |
| 击杀 14/22/30 过关 | 探索到 Boss 房击杀 Boss 过关 |
| 场间回血 20% | 地图上有治愈泉格子 |
| 难度倍率 1.0/1.2/1.5/2.0 | 怪物基础数值随距离起点的格数递增 |
| 60s 最低存活时间 | 无时间限制 |

### 6.2 难度缩放

怪物部署在距离起点 N 格处时，获得难度加成：
- 0-10 格：×1.0（新手区）
- 11-20 格：×1.3
- 21-30 格：×1.6
- 31+ 格：×2.0
- Boss 房内：×2.5

基于 A* 路径距离到起点，非直线距离。

### 6.3 Boss

- Boss 固定在地图右上角的 Boss 房间中
- 玩家需要先探索到 Boss 房间附近
- 进入 Boss 房间前有"Boss 门"格，经过时显示 Boss 预览
- Boss 房间内无格子效果（纯战斗）
- Boss 仍可召唤小怪（出现在 Boss 房间内空格上）
- 击杀 Boss → 通关

### 6.4 失败

- 英雄 HP ≤ 0 → 失败
- 可重新开始（重新生成地图）或回主菜单

## 7. 保留的系统

### 7.1 候选/手牌
- 3 选 1 + 2s CD + 全手牌刷新：完全保留
- NextHandPreview：完全保留
- 卡池权重：完全保留

### 7.2 怪物机制
全部保留，适配格子：

| 怪物 | 机制适配 |
|------|----------|
| 史莱姆 | 死亡分裂：小史莱姆出现在周围空格（1-2格内随机） |
| 蝙蝠 | 飞行：经过负面格不触发效果（英雄经过时仍触发） |
| 狼 | 群族：4 格范围内有同族存活 → 攻速 +50% |
| 哥布林 | 死亡爆炸：2 格范围内所有单位受 8 伤 |
| 骷髅 | 墓碑：死后留墓碑在原格，需再次击杀 |
| 石像鬼 | 防御光环：3 格范围内友方怪 +2 防 |
| 毒蛇 | 死亡毒池：所在格变为"毒沼"格（永久） |

### 7.3 共鸣/被动
- 21 个被动 + 8 混合：完全保留
- Tier II/III 击杀召唤范围效果 → 改为：击杀后将周围 1-2 格空地转化为对应效果格（持续 8s 后恢复空地）

### 7.4 Combo
- 生态专精/异种联动/密集部署：完全保留
- 效果不变

### 7.5 击杀属性
- 击杀累计属性：完全保留
- 精英变体 ×2：保留

## 10. 策略深度分析

### 玩家每次部署时需要思考：

1. **选哪张**（与 v5 相同）
   - 哪条共鸣路径需要推进？
   - 能触发什么 combo？
   - 精英版值不值得走远路去打？

2. **放在哪**（大幅增强）
   - 英雄到该格的路径经过哪些格子？
   - 路径上有治愈泉还是陷阱？
   - 放在迷雾边缘可以探索新区域吗？
   - 放得近（路径短、安全）还是放得远（路径长、格子效果多）？
   - 需要绕路去踩宝箱吗？
   - Boss 方向应该怎么接近？

3. **路径规划**（全新）
   - 场上还有其他未击杀的怪，击杀当前目标后英雄会走向谁？
   - 这个顺序经过的格子效果如何？
   - 要不要先部署一个"中继点"怪物引导英雄绕开负面区域？

### 风险-收益权衡：

- **探索 vs 安全**：走向迷雾边缘 = 可能发现宝箱也可能踩陷阱
- **远路 vs 近路**：远路经过更多正面格但也更多负面格
- **冲 Boss vs 养成**：直奔 Boss 省时间但英雄可能太弱
- **中继怪 vs 直奔目标**：有时部署一只"路标怪"引导英雄绕路更值

## 11. 一局游戏节奏（约 8-12 分钟）

```
起步（0-2 分钟）
├── 英雄在起点，周围已探索
├── 部署怪物探索周围房间
├── 踩到正面格积累优势
├── 首个共鸣 Tier I 解锁
│
中期（2-6 分钟）
├── 探索 2-3 个房间
├── 遭遇负面格/问号格
├── 怪物难度开始提升
├── 共鸣 Tier II 解锁，被动变强
├── 通过路径规划绕开负面区域
├── 发现传送阵/视野塔等特殊格
│
后期（6-10 分钟）
├── 接近 Boss 区域
├── 负面格密度增加，每一步都是赌博
├── 共鸣 Tier III / 混合共鸣解锁
├── 在 Boss 门前做最后准备
│
Boss 战（10-12 分钟）
├── 进入 Boss 房间
├── Boss 战（继续部署怪物获取属性/被动效果）
├── 击杀 Boss → 通关
```
