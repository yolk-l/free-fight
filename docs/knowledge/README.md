### v5 速查（详见 09）

- 玩家唯一操作 = 3 选 1 + 部署位置（无主动技能/弃牌/装备）
- 部署 CD 2s，CD 结束自动刷新候选；候选 10% 概率精英
- 战场地形：6 种 × 4 个固定位置（Boss 战不生成）
- 怪物 7 种独特机制：分裂/飞行/群族/爆炸/墓碑/光环/毒池
- 共鸣 7 路径 × 3 阶 = 21 个被动技能 + 8 混合机制
- Combo 模式识别：生态专精 / 异种联动(5 个) / 密集部署
- 击杀属性自动累计（按怪种映射，跨场保留）
- Run 4 场：12/18/24 杀 + 60s + 第 4 场 Boss
